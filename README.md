##### 目前已有的功能:
* 1.登录，注册
* 2.自动登录，退出登录，单点登录
* 3.添加好友:1.扫一扫添加好友 2.通过账号(手机号)添加好友
* 4.会话列表
* 5.聊天
* 6.发送表情，文字，图片，语音，位置
